Allow user to attach meta data about a DIY project above their post's content area.

== Description ==

Displays a meta box containing information about a DIY project above the post content

**Development of this plugin is done [on GitHub](https://github.com/Magnacarter/diy-posts-plugin). Pull requests welcome. Please see [issues](https://github.com/Magnacarter/diy-posts-plugin/issues) reported there before going to the plugin forum.**
